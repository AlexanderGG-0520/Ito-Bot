# Ito Bot 利用規約

- **運営者:** Alec
- **制定日:** 2026年7月21日
- **最終更新日:** 2026年7月21日

## 1. 適用範囲

本規約は、Alecが運営するDiscord Bot「Ito Bot」（以下「本Bot」）の利用、関連するGitHub Pagesサイト、および本Botに関するドキュメントに適用されます。

## 2. 本規約への同意

本Botを招待、利用、または本Botとインタラクションすることにより、本規約に同意したものとします。同意できない場合は、本Botを利用しないでください。

本Botの利用にあたっては、[Discord利用規約](https://discord.com/terms)、[Discordコミュニティガイドライン](https://discord.com/guidelines)、利用するサーバーの規則、および適用される法令にも従う必要があります。

## 3. サービス内容

本Botは、Discordのスラッシュコマンド、Gateway、およびAPIを利用して、協力型のカードゲーム「ito」に着想を得たゲーム機能を提供します。ロビー、テーマ選択、非公開の手札表示、宣言、カードプレイ、ステージ進行などの機能が含まれます。

本Botの機能や仕様は、保守、Discord APIの変更、セキュリティ、その他の事情により変更される場合があります。継続的な提供、特定の応答時間、またはすべての環境との互換性を保証するものではありません。

## 4. 利用条件

利用者は、Discordが認めるアカウントを使用し、本規約、Discordの規約、利用するサーバーの規則、および適用される法令に従って本Botを利用するものとします。利用者のアカウントを通じて行われた操作については、利用者が責任を負います。

## 5. 禁止事項

利用者は、次の行為をしてはなりません。

- 法令、公序良俗、Discordの規約、または第三者の権利に違反する行為
- 本Botを利用した嫌がらせ、脅迫、詐欺、なりすまし、その他の有害な行為
- 意図的な大量コマンド、スパム、自動化による過度な利用、または本Botに不必要な負荷をかける行為
- 本Botまたは運用環境への不正アクセス、アクセス制御の回避、脆弱性の悪用、またはサービスの妨害
- ログ、ヘルスチェック、分析、セキュリティ対策、コマンド登録などの運用を妨害する行為
- Bot Token、認証情報、シークレット、アクセストークンなどを取得、公開、窃取、交換、または不正利用する行為
- 他の利用者の非公開の手札、ゲーム状態、アカウント、またはインフラに無断でアクセスする行為
- Alec、Discord、GitHub、ゲームの権利者その他の者になりすます行為

公開されているソースコードを通常の目的で閲覧することや、適用されるライセンスに従って利用すること自体は禁止されません。ただし、悪意のある侵害や脆弱性の悪用を目的とする分析は禁止します。

脆弱性を発見した場合は、[セキュリティポリシー](https://github.com/AlexanderGG-0520/Ito-Bot/security/policy)を確認してください。現在の`SECURITY.md`には非公開の報告窓口や機密情報を送る手順が明記されていないため、公開Issueに認証情報、攻撃手順、実行可能な概念実証コード、その他悪用につながる情報を投稿しないでください。公開Issueを利用する場合も、機密情報を含まない概要にとどめてください。

## 8. ゲーム状態とデータ消失

本Botのゲーム状態は現在、Botプロセスのメモリ上だけで管理されており、ゲーム進行を保存する永続的なデータベースは使用していません。プロセス、Pod、Deployment、Kubernetes Node、または関連インフラの再起動、クラッシュ、再配置、交換、障害、保守などにより、進行中のゲーム、手札、宣言、ライフ、ステージ、テーマ、進行状況、結果が失われる場合があります。

`/end`は対象のゲーム状態を削除し、`/leave`は参加状態を更新します。本Botをゲーム結果や進行状況の恒久的な保存先として利用しないでください。
